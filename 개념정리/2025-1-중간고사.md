### 1. 기본적인 class 구조

<자바 프로그램의 전체구조를 작성>
```java
public class Main{
    public static void main(String[] args){
        System.out.print("Hello, Java!");
    }
}
```
- public: 공개범위- 완전공개
- static: 객체 생성 여부 - 객체 없이도 가능
- void: 함수 반환값 여부 - 반환값 없는 유형
- String[]: 문자열 저장장소
